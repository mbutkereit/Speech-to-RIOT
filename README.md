# Speech-to-RIOT
SS 2017 - RIOT im Internet of Things - Bachelor Project of Computer Science and Electrical Engineering
Voice control for IOT-devices

# Required Hardware
- 1x Raspberry Pi
- 1x Microphone (USB)
- 1x Speakers (AUX)
- x sensors
